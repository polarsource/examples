# Getting started with Polar and Next.js 

## Clone the repository

```bash
npx degit polarsource/examples/with-nextjs ./with-nextjs
```

## How to use

1. Run the command below to copy the `.env.example` file:

```
cp .env.example .env
```

2. Run the command below to install project dependencies:

```
npm install
```

3. Run the Next.js application using the following command:

```
npm run dev
```
