# Getting started with Polar, Supabase and React Router v7

## Clone the repository

```bash
npx degit polarsource/examples/with-react-router-supabase ./with-react-router-supabase
```

## How to use

1. Run the command below to copy the `.env.example` file:

```
cp .env.example .env
```

2. Run the command below to install project dependencies:

```
npm install
```

3. Run the React Router application using the following command:

```
npm run dev
```
