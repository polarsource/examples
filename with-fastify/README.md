# Getting started with Polar and Fastify

## Clone the repository

```bash
npx degit polarsource/examples/with-fastify ./with-fastify
```

## How to use

1. Run the command below to copy the `.env.example` file:

```
cp .env.example .env
```

2. Run the command below to install project dependencies:

```
npm install
```

3. Run the Fastify application using the following command:

```
npm run dev
```
