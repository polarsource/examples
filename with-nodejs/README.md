# Getting started with Polar and Node.js

## Clone the repository

```bash
npx degit polarsource/examples/with-nodejs ./with-nodejs
```

## How to use

1. Run the command below to copy the `.env.example` file:

```
cp .env.example .env
```

2. Run the command below to install project dependencies:

```
npm install
```

3. Run the Node.js application using the following command:

```
npm run dev
```
