# Getting started with Polar and Express v4

## Clone the repository

```bash
npx degit polarsource/examples/with-express-v4 ./with-express-v4
```

## How to use

1. Run the command below to copy the `.env.example` file:

```
cp .env.example .env
```

2. Run the command below to install project dependencies:

```
npm install
```

3. Run the Express application using the following command:

```
npm run dev
```
