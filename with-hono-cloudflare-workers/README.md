# Getting started with Polar, Hono and Cloudflare Workers

## Clone the repository

```bash
npx degit polarsource/examples/with-hono ./with-hono
```

## How to use

1. Run the command below to install project dependencies:

```bash
npm install
```

2. Configure environment variables:

Update the `wrangler.jsonc` file in the project root with your environment variables.

3. Run the Next.js application using the following command:

```bash
npm run dev
```
