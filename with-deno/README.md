# Getting started with Polar and Deno

## Clone the repository

```bash
npx degit polarsource/examples/with-deno ./with-deno
```

## How to use

1. Run the command below to copy the `.env.example` file:

```
cp .env.example .env
```

2. Run the command below to install project dependencies:

```
deno install
```

3. Run the Elysia application using the following command:

```
deno task dev
```
