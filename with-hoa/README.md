# Getting started with Polar and Hoa

## Clone the repository

```bash
npx degit polarsource/examples/with-hoa ./with-hoa
```

## How to use

1. Run the command below to copy the `.env.example` file:

```
cp .env.example .env
```

2. Run the command below to install project dependencies:

```
npm install
```

3. Run the Hoa application using the following command:

```
npm run start
```
